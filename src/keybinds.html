Navigation

h
Move cursor left
j
Move cursor down
k
Move cursor up
l
Move cursor right
w
Jump forward to start of word
b
Jump backward to start of word
gg
Jump to top of document
G
Jump to bottom of document
e
Jump forward to end of word
ge
Jump backward to end of word
0
Go to beginning of line
^
Go to first non‑blank character
$
Go to end of line
g_
Go to last non‑blank character
{
Jump to previous paragraph
}
Jump to next paragraph
x
delete backwards
Ctrl + b
Page up
Ctrl + f
Page down
←
Move cursor left (or move word backward with Ctrl)
→
Move cursor right (or move word forward with Ctrl)
↑
Move cursor up (or jump to previous paragraph with Ctrl)
↓
Move cursor down (or jump to next paragraph with Ctrl)
.
repeat the last motion

Text Objects

iw
“Inner word”: selects the word without surrounding spaces
aw
“A word”: selects the word along with any adjoining whitespace
diw
Delete inner word
daw
Delete a word
ciw
Change inner word
yaw
Yank a word

Modes

i
Enter insert mode
a
Append (insert after cursor)
A
Append at end of line
I
Insert at beginning of line
v
Enter visual mode
V
Enter visual line mode
Esc
Return to normal mode

Editing (Operators & Commands)

d + motion
Delete text over a motion
dd
delete the whole line
c + motion
Change text over a motion
y + motion
Yank (copy) text over a motion
yy
yank whole line
p
Paste
u
Undo
r
Redo
o
Open new line below and insert
O
Open new line above and insert
X
Delete the character before the cursor

Insert Mode Commands

Ctrl + h
Delete character before cursor
Ctrl + o
Temporary normal mode from insert mode
Esc
Exit insert mode

Visual Mode Commands

d
(In visual mode, used for deleting the selected text)
Esc
Exit visual mode
y
Yank (copy) selection
p
Paste over selection
u
Undo
r
Redo

Special Line Commands

cc
Change (cut and switch to insert mode) the current line(s)
C
Equivalent to “c$”: change from the cursor to the end of the line
D
Equivalent to “d$”: delete from the cursor to the end of the line
Y
Equivalent to “y$”: yank from the cursor to the end of the line
P
Paste before the cursor

Numeric Prefix

Numbers
A numeric count (e.g., 2, 3, etc.) can be prefixed to motions or operators to repeat the command multiple times (for example, 2w, 3dd, or 4yy).
